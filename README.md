# Viajes Viajaja – Landing Page

Landing page responsiva para la agencia de viajes **Viajes Viajaja**, enfocada en presentar su propuesta de valor, servicios principales y canales de contacto.

## Tecnologías

* React + TypeScript
* Tailwind CSS
* Vite
* Despliegue en **Vercel**

## Descripción

La interfaz está diseñada como una página de una sola vista (*single-page*), organizada en secciones bien definidas:

* Hero (presentación principal)
* Quiénes somos
* Servicios
* Categorías de viaje
* Trámites especiales
* Testimonios
* Razones para elegir la agencia
* Llamado a la acción final

El diseño prioriza la legibilidad, una estructura clara y la adaptación a dispositivos móviles y de escritorio, manteniendo una estética sobria y profesional adecuada para un portafolio de servicios turísticos.

## Ejecución local

1. Clonar este repositorio:

```bash
git clone https://github.com/cantor11/viajes_viajaja_landing_page.git
cd mf_travel_landing_page
```

2. Instalar dependencias:

```bash
npm install
```

3. Iniciar el entorno de desarrollo:

```bash
npm run dev
```

4. Abrir en el navegador la URL indicada en la consola.

## Build

Para generar la versión de producción:

```bash
npm run build
```

Los artefactos generados pueden desplegarse en **Vercel** u otro servicio de hosting estático compatible.
